DATABASE SETUP OPTIONS:
Three SQL_SETUP.md provided with complete schemas and code for:

1. PostgreSQL (Recommended for production)
2. MySQL
3. SQLite (Good for development/testing)

HOW TO ADD DATABASE:

Step 1: Choose your database and create table using SQL_SETUP.md

Step 2: Install database client
  pnpm add pg          # for PostgreSQL
  pnpm add mysql2      # for MySQL
  pnpm add better-sqlite3  # for SQLite

Step 3: Add DATABASE_URL to .env
  DATABASE_URL=postgresql://user:pass@localhost:5432/axifleetx

Step 4: Replace server/routes/contact.ts with PostgreSQL/MySQL/SQLite version from SQL_SETUP.md

Step 5: (Optional) Add email notifications using nodemailer

Step 6: Test the form - it will now save to database

TESTING THE FORM:
1. Click any CTA button ("Get Started", "Watch Demo", etc.)
2. Fill in the form fields
3. Click "Send Message"
4. See success message
5. Check server logs to verify submission

API ENDPOINT:
POST /api/contact
Content-Type: application/json

Request body:
{
  "name": "string (required)",
  "email": "string (required)",
  "phone": "string (optional)",
  "company": "string (optional)",
  "message": "string (required)"
}

Response:
{
  "success": true,
  "message": "Thank you for your interest. We'll contact you soon.",
  "id": 1  // database ID (when database is added)
}
